# Musculoskeletal Posture & Gait Assistant

앉은 자세와 걷는 행동(보행)을 분석해서 근골격계 위험도를 추정하고, 맞춤형 예방 운동을 추천하는 프로그램입니다.

## 기능
- 웹캠에서 사람의 포즈(랜드마크) 실시간 추정
- 앉은 자세 분석
  - 거북목(목 전방 자세) 위험
  - 허리 과굴곡/과신전 위험
  - 골반 정렬 문제 추정
- 보행 분석
  - 좌우 무릎 굴곡 비대칭
  - 골반 좌우 흔들림(Trendelenburg 의심)
  - 몸통 기울기 불안정
- 위험도 점수(0~100) 및 등급(낮음/중간/높음) 제공
- 위험군별 예방 운동 루틴 제안

## 주의
- 본 프로그램은 **의료 진단 도구가 아니며**, 교육/예방 목적의 참고용입니다.
- 통증, 저림, 마비, 보행 장애가 있으면 정형외과/재활의학과/물리치료 전문가 상담이 필요합니다.

## 설치
```bash
pip install -r requirements.txt
```

## 실행
```bash
python main.py --mode sitting
```
또는
```bash
python main.py --mode gait
```

## Streamlit 웹 실행
브라우저에서 보기 쉬운 대시보드를 사용하려면:
```bash
streamlit run streamlit_app.py
```
실행 후 표시되는 로컬 주소(예: `http://localhost:8501`)로 접속하면 됩니다.

### Streamlit에서 결과 저장
- `현재 결과 저장`: 현재 분석 결과 1건 저장
- `기록 초기화`: 저장 기록 삭제
- `CSV 다운로드`: 누적 저장 기록을 CSV 파일로 다운로드

### 실행 옵션
- `--mode sitting|gait`
- `--camera 0` (기본 카메라 인덱스)
- `--window 120` (gait 모드에서 분석 프레임 길이)

## 사용 방법
1. 프로그램 실행 후 카메라 앞에 전신이 보이게 위치합니다.
2. `sitting` 모드: 평소처럼 앉아 자세를 유지합니다.
3. `gait` 모드: 카메라 앞에서 제자리 걷기 또는 짧은 보행을 반복합니다.
4. 화면 좌측 상단에 위험도와 주요 코멘트가 표시됩니다.
5. 터미널에는 권장 운동 루틴이 주기적으로 출력됩니다.

## 파일 구조
- `main.py`: 웹캠 입력 및 UI/실행 루프
- `analyzer.py`: 자세/보행 분석 및 운동 추천 로직
- `pose_utils.py`: 각도/거리/대칭 계산 유틸리티
